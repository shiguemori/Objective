## Jogo Gourmet Replica
### O sistema foi desenvolvido puramente em PHP a partir do zero
#### Requisitos:
Ter o PHP-8.1 instalado na maquina 

#### Como rodar
Execute o composer
``composer install``
e uma das formas é ir na raiz do projeto rode o comando:
``php -S localhost:8000``
